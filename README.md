# WEB1Aula03

Nesta aula, aprendemos sobre estilizar os textos na página inicial para o usuário, como: destacar um parágrafo, arrastar parágrafos, mudar cor de palavras para dar destaque, mostrar a definição de uma palavra quando coloca o mouse sobre ela e a inserir imagens.
